# APK V0.3.8 智能提醒验收

## 结论

`PASS`。智能提醒与固定日期两种模式均可选择、保存和重新加载；智能模式只在食品输入或相关状态变化时，根据当时的剩余时长、保存方式、食品类别与开封后期限生成一次可解释的提醒快照，日期自然流逝不重算。固定模式只使用用户填写的提前天数。`expiryDate` 仍是最终可食用日期和提醒计算的基准。

## 边界

智能规则只用于调整查看和处理提醒频率，不判断食品是否绝对安全，也不替代用户对包装标签和实际状态的确认。
