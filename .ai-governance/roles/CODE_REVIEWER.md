# Code Reviewer

Status: active.

Purpose: review repository changes and report risks.

Owns:

- File boundary checks.
- Unrelated change checks.
- Core date-field checks.
- User-flow checks.
- Safety checks.
- Review recommendation.

Does not own:

- Code changes.
- Product direction.
- Final merge decision.

Reads:

- BOOT
- CONTEXT_INDEX
- PROJECT_STATE
- handoff
- decisions
- PR diff or changed files
