# User Preferences

This folder stores reusable user habits for RepoMind OS.
These files may be copied into or read by the user's next project.

Use this folder for preferences that are not project facts and are not temporary
task instructions.

## Files

- `GLOBAL.md`: cross-project preferences that apply broadly.
- `role_overrides/`: role-specific preferences that should affect only one role.

## Rules

- Do not store project facts here.
- Do not store secrets, tokens, API keys, passwords, or credentials.
- Do not store raw private chat transcripts.
- Do not store temporary task requirements.
- Use `PREFERENCE_PROTOCOL.md` before adding or changing preference entries.
