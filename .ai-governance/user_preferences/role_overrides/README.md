# Role Preference Overrides

This folder stores long-lived user preferences that apply to one RepoMind OS
role only.

Use role overrides when a preference would be too broad or misleading in
`user_preferences/GLOBAL.md`.

## Rules

- Keep entries role-specific.
- Do not store project facts.
- Do not store secrets or raw private chat transcripts.
- Do not use these files for temporary task instructions.
- Use `PREFERENCE_PROTOCOL.md` before adding or changing entries.
